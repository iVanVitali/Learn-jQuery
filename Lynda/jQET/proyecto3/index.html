Proyecto3 - Crear/Cambiar el contenido con jQuery!

jQuery hace que sea muy fácil crear nuevo contenido en la página y cambiar el contenido existente de la página.

Creación del contenido

Crear un nuevo contenido es tan simple como pasar una cadena de HTML a la función $().

Por ejemplo, $("<p>") crea una nueva etiqueta de párrafo vacía. Para agregar contenido nuevo al párrafo, puede escribir simplemente $("<p>").append("<em>Hello there</em>").

Cambiar el contenido

Hay varias maneras para cambiar el contenido de la página usando jQuery, dependiendo de sus necesidades.

Las funciones html() y text() pueden usarse para manipular directamente el contenido de los elementos, Y hay formas de controlar cómo se puede insertar y mover contenido en la página.
